## 🚀 Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/Harsh18v/<repo-name>.git
cd <repo-name>
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view it.
